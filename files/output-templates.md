# Output Templates

Use these templates when producing CVI adaptation deliverables. Three outputs are typically needed: student-ready materials, teacher instructions, and a fidelity checklist.

---

## Student-Ready Material Standards

Before finalizing any student materials, verify these requirements:

### Universal Requirements Checklist

- [ ] **Learning goal preserved** - Same academic content and rigor as original
- [ ] **Visual complexity reduced** - Appropriate for student's Phase
- [ ] **Spacing increased** - Minimum 1.5x standard; often 2x for Phase I-II
- [ ] **Decorative elements removed** - No borders, clip art, or non-essential images
- [ ] **Consistent visual language** - Same fonts, icons, and layout patterns throughout
- [ ] **Field preference honored** - Materials positioned in student's functional visual field
- [ ] **Contrast appropriate** - High contrast unless assessment indicates otherwise
- [ ] **One direction per section** - Clear, single focus for each page or segment
- [ ] **Anchors provided** - Clear starting points and visual markers as needed

### Format-Specific Standards

**Print Materials:**
- Matte paper (no gloss)
- Font size per functional vision assessment (minimum 14pt; often 18-24pt)
- Single column preferred
- Page numbers clear and consistent
- Binding allows pages to lie flat

**Digital Materials:**
- Zoom and reflow enabled
- Compatible with screen reader if needed
- Dark mode option available
- Adjustable font size and spacing
- No auto-playing media

**Projected Materials:**
- High contrast (dark text on light OR light text on dark)
- Large font (minimum 28pt for projected slides)
- Minimal text per slide (6 lines maximum)
- Simple backgrounds (no photographs or busy patterns)
- Personal copy provided to student at near

## Quick Documentation Format

For informal or rapid documentation when full templates are not needed:

```
[Date] | [Material] | [Student]
Phase: [X] | Profile: [X]
Changes made: [Brief list]
Result/Observations: [What worked, what to adjust]
Next steps: [Any follow-up needed]
```

**Example:**
```
2024-01-15 | Ch. 4 Science Worksheet | Maria S.
Phase: II | Profile: Dorsal + Lower-level
Changes made: Enlarged to 18pt, reduced to 3 problems/page, added boxes, provided masking strip
Result/Observations: Completed all problems; no fatigue signs; asked to "show more" after 2nd page
Next steps: Try 4 problems/page next time; maintain other supports
```
