# CVI Content Adaptation Intake Checklist

## Student access
- CVI Range Phase: I / II / III (or unknown)
- Processing profile (if known): dorsal / ventral / lower-level / higher-VFD / mixed
- Micro-characteristics:
  - Complexity sensitivity (clutter/crowding)
  - Field preference
  - Latency
  - Need for movement
  - Light attraction/sensitivity
  - Distance viewing difficulty
  - Fatigue/variability

## Material + task
- Subject/goal
- Content type (worksheet, slides, assessment, diagram, etc.)
- Delivery (print, projected, digital)
- Time constraints and environment

## Success definition
- What “access” means here (accurate responses? independence? endurance? speed?)
- What must not change (standards, item wording, scoring)
